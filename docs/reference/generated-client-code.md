# Generated Client Code

Generated clients are contract artifacts. Treat their source as output, not hand-maintained application code.

## Rules

- Do not edit generated client source files directly.
- Prefer fixing the source contract, generator configuration, or generator version.
- Do not add local template overrides only to silence warnings.
- Consider a local template override only when generated code blocks compilation, runtime behavior, or a required platform upgrade.
- Keep generated-code warnings separate from application-code regressions during review.
